# Índice de Specs

Este índice centraliza as especificações do WellWave e ajuda a navegar por contexto.

## Principais specs

- `specs/1-wellwave-mvp/` — escopo do MVP, plano e tarefas
- `specs/wellwave-platform/` — visão da plataforma e evolução
- `specs/inslate-backend/` — backend e integrações

## Templates

- `specs/templates/feature-spec.md`
- `specs/templates/api-spec.md`
- `specs/templates/technical-design.md`
- `specs/templates/deployment-guide.md`

## Como usar

1. Crie uma pasta em `specs/` para a nova feature.
2. Inicie com `spec.md` usando um dos templates.
3. Atualize este índice com a nova spec.
