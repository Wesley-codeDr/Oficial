# Performance

Diretrizes de performance alinhadas ao uso em emergencia.

## App Router e RSC

- Server Components por padrao.
- Client Components apenas quando necessario.

## Cache e dados

- Cache de dados do Next.js quando aplicavel.
- Evitar recomputacao de narrativas sem necessidade.

## Frontend

- Bundle reduzido com imports seletivos.
- Componentes pesados com lazy loading quando fizer sentido.
