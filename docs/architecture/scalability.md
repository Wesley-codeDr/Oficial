# Scalability

Estrategia inicial para crescer sem perder confiabilidade.

## Aplicacao

- Escala horizontal via Vercel.
- Separar workloads pesados (IA) quando necessario.

## Dados

- Indexacao em tabelas criticas.
- Monitorar consultas lentas e ajustar.

## Observabilidade

- Sentry para erros e performance.
- Logs estruturados para diagnostico.
